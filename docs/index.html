“Ein Hackathon-Variante zur Evaluation der Klickdaten des KI-Tools ‘HaNS’”

Autor:in

Sebastian Sauer

Veröffentlichungsdatum

8. Dezember 2025

1 Willkommen zum Projekt “Hans-Hackathon”!

Come to the HaNS-Side: we have videos.

1.1 Gegenstand und Ziel

In diesem Modul analysieren Sie die Klickdaten zum KI-Tool HaNS.

Ziel ist es, das Tool HaNS zu evaluieren. Die Klickdaten sollen genutzt werden, um den Nutzen des Tools für die Studierenden besser einschätzen zu können. Darüber hinaus soll verstanden werden, wie Studierende das Tool zum Lernen nutzen, und wie das Tool verbessert werden kann, um das Lernen noch erfolgreicher zu gestalten.

1.2 Kursaufbau und Didaktik

Der grundlegende Aufbau ist, dass den Studierenden jede Woche eine “Challenge” (“Herausforderung”, im Sinne eines Aufgabenpakets) präsentiert wird, die bis zur nächsten Woche gelöst werden soll.

Im Unterricht wird Material, das zur Lösung der Aufgabe nötig ist, besprochen oder es wird auf das Material verwiesen. In der folgenden Stunde stellt die Lehrkraft dann eine Musterlösung vor und bespricht die von den Studierenden eingereichte Lösung. Dann wird wiederum eine neue Challenge angegangen usw.

1.3 Modulverlauf

Der Modulverlauf gibt einen Überblick, welches Thema in welcher Woche bzw. wann behandelt wird. Pro Woche wird ein Thema behandelt.

Tipp

Es ist nützlich für Sie, den Modulverlauf immer mal wieder zu konsultieren, damit sie wissen, welche Themen als nächstes behandelt werden. \(\square\)

1.4 Hinweise

Bitte beachten Sie, dass sich eine bestimmte Challenge noch ändern kann, solange sie nicht begonnen hat. Beachten Sie auch die Prüfungshinweise und sonstige Hinweise im Hinweisbuch.

1.5 Vorbereitung

Bitte stellen Sie sicher, dass Sie eine aktuelle Version von R auf Ihrem Computer am Laufen haben. Bitte bringen Sie einen Computer (mit Internetanschluss) in jede Stunde mit. Als Oberfläche (IDE) für R bietet sich neben RStudio auch die IDE von Posit an: Positron.

1.6 Werkzeuge

  • R (Programmiersprache)
  • RStudio (IDE für R)
  • Positron (IDE für R, Alternative zu RStudio)
  • Quarto (für Berichte) - schon vorinstalliert in RStudio und Positron
  • Git (Versionsverwaltung) - optiona
  • GitHub (Online-Plattform für Git-Repositories) - optional
  • Tidyverse (R-Paket-Sammlung für Datenanalyse)
  • Vim (Texteditor) - optional; VIM-Modus in RStudio und Positron ist auch möglich; Liebeserklärung an VIM
  • Obsidian (Notizen) - optional
  • AI-Bots wie Claude, ChatGPT, Bard, etc. - optional, aber empfohlen

1.7 Nutzen Sie KI-Dienste

Nutzen Sie KI-Dienste wie ChatGPT, Claude, Bard, etc., um Ihre Produktivität zu steigern. Diese Dienste können Ihnen helfen, schneller zu programmieren, Fehler zu finden, Dokumentation zu verstehen, und vieles mehr. Seien Sie kreativ und probieren Sie verschiedene Dienste aus, um herauszufinden, welcher Dienst am besten zu Ihnen passt. IDEen wie RStudio und Positron haben oft auch Plugins oder Erweiterungen, die KI-Dienste integrieren.

Viel Erfolg und Spaß im Kurs!
